# LAN和WAN的区别？

当计算机或其他设备在同一地理区域内互相连接时，这就形成了一个局域网（Local Area Network，LAN）。局域网通常被用于连接公司、学校或者家庭内部的设备。因为设备之间的距离比较近，因此局域网的传输速度比较快，可以使用各种协议和技术，如Ethernet、Wi-Fi等。

而广域网（Wide Area Network，WAN）则是指跨越较大地理区域的网络，可以覆盖国家、洲甚至全球。它是由许多局域网连接而成的，通常由电信公司、互联网服务提供商等机构负责维护。由于广域网中设备之间的距离比较远，因此传输速度可能会较慢，并需要使用更加高级的协议和技术，如TCP/IP协议、光纤通信、卫星通信等。


# 四种网络的区别

表一：

| 类型 | 简介                                                         | 区别                                                         |
| ---- | ------------------------------------------------------------ | ------------------------------------------------------------ |
| LAN  | 局域网 (Local Area Network)，是一种覆盖范围较小的网络，通常在一个建筑物或者一个校园内部使用。 | 传输速度较快，设备间使用有线连接。                           |
| WLAN | 无线局域网 (Wireless Local Area Network)，也称为Wi-Fi，是一种使用无线信号传输数据的网络，可以覆盖范围较大的区域。 | 传输速度相对较慢，但更加灵活，可以通过无线信号连接设备。     |
| WAN  | 广域网 (Wide Area Network)，是一种连接不同城市、不同国家、甚至不同大陆的计算机和网络设备的网络。 | 覆盖范围更广，设备间通常通过互联网等公共网络连接。           |
| WWAN | 无线广域网 (Wireless Wide Area Network)，是一种无线网络技术，可以在全球范围内进行数据传输。 | 覆盖范围最广，可以通过移动网络进行数据传输，但传输速度相对较慢。 |

表二：

| 类型 | 全称                        | 解释                                   | 范围                 | 传输介质   |
| ---- | --------------------------- | -------------------------------------- | -------------------- | ---------- |
| LAN  | Local Area Network          | 局域网，指的是小范围内的网络           | 办公室、学校、家庭等 | 有线或无线 |
| WLAN | Wireless Local Area Network | 无线局域网，指通过无线信号传输的局域网 | 办公室、学校、家庭等 | 无线       |
| WAN  | Wide Area Network           | 广域网，指连接较大地理范围的网络       | 省、国家、全球等     | 有线或无线 |
| WWAN | Wireless Wide Area Network  | 无线广域网，指通过无线信号传输的广域网 | 省、国家、全球等     | 无线       |
